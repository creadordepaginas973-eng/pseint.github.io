ALGUNAS INSTRUCCIONES DE CODIGO BASICAS

En programacion tambien hay instrucciones basicas indispensables al momento de programar las cuales son las siguientes

DEFINIR

Esta es la instruccion mas basica ya que es la que indica la creacion de una variable por asi decirlo.
UN EJEMPLO SERIA: Definir numero Como Entero

ESCRIBIR

Esta instruccion se utiliza para mostrar un mensaje en pantalla,pero eso si, para que se muestren deben estar escrito dentro de comillas ya sean comillas dobles o comillas sencillas,tambien sirve para que se escriban los valroes de alguna variable,pero para que aparezca se debe poner el nombre de la variable sin comillas,y se le debe poner una coma antes del nombre siempre y cuando se haya escribido antes de la misma

Ejemplos

Escribir "hola"

Escribir "El resultado es",resultado

LEER

Esta instruccion se utiliza para que al momento de la ejecucion el usuario le pueda poner el valor que el desee aunque tambien depende como se haya definido, si se definio como Entero o Real solo aceptara numero,si se definio como caracter aceptara solamente 1 sola letra, mientras que si se definio como cadena

Ejemplos

Leer numero

Leer nombre
