VARIABLES:

EN PSEINT EXISTEN VARIOS TIPOS DE DATOS EN LOS QUE SE PUEDEN ALMACENAR VARIABLES, Y SI TE PREGUNTAS QUE ES UNA VARIABLE?, PUES EN PROGRAMACION VENDRIA SIENDO EL LUGAR DONDE SE ALMACENAN O SE GUARDAN LOS DATOS QUE SE PIDAN O SE AGREGUEN AL MOMENTO DE EJECUTAR O PROBAR EL CODIGO Y EXISTEN DIFERENTES TIPOS DE DATOS LOS CUALES SE EXPLICAN A CONTINUACION.

ENTERO

EN PSEINT UNO DE LOS VALORES QUE SE LE PUEDE AGREGAR A UNA VARIABLE ES EL ENTERO, COMO SU NOMBRE LO INDICA SOLAMENTE ACEPTA NUMEROS ENTEROS (NADA DE DECIMALES),ESTO ES INDISPENSABLE AL MOMENTO DE HACER CALCULOS DE OBJETOS O AL HACER OPERACIONES MATEMATICAS (MAYORMENTE PARA SUMAS RESTAS O MULTIPLICACIONES) Y TAMBIEN PARA PODER ELEGIR OPCIONES CON LA ESTRUCTURA SEGUN O PARA DARLE INSTRUCCIONES AL CICLO PARA O INCLUSO PARA DAR CONDICIONES CON LA ESTRUCTURA SI SINO.

REALES

TAMBIEN SE ENCUENTRAN LOS DATOS DE TIPO REAL,ESTE VALOR ES CASI PARECIDO A LOS ENTEROS CON LA UNICA DIFERENCIA DE QUE ACEPTA LOS NUMEROS DECIMALES Y QUE NO SE PUEDE OCUPAR CON EL CICLO PARA NI CON LA ESTRUCTURA SEGUN.

CARACTER

EN PSEINT TAMBIEN EXISTEN LOS DATOS DE TIPO CARACTER QUE SON LOS QUE ACEPTAN 1 UNICA LETRA,NO MAS,NI TAMPOCO NUMEROS SE OCUPA PARA PODER ESCOGER UNA OPCION CON LA ESTRUCTURA SI SINO O PARA HACER ALGUNA INSTRUCCION EN ESPECIFICO.

CADENA

PARA FINALIZAR TAMBIEN ESTAN LOS DE TIPO CADENA DE TEXTO,LOS CUALES SE ENCARGAN DE ALMACENAR MAS DE 1 LETRA, LO CUAL ES IDEAL PARA PODER GUARDAR PALABRAS COMPLETAS,NOMBRES DE PERSONAS,PAISES,O DE CUALQUIER OBJETO YA SEA EL CASO,ESO SI, ESTO NO ACEPTARA NINGUN TIPO DE VALOR NUMERICO.
